# Neural Network Architectures

This repository contains implementations of fundamental and advanced neural network architectures in Python.

---

## 📚 Overview

The goal of this project is to study and implement different types of neural networks from scratch in order to better understand how they work internally.

---

## 🧠 Implemented Models

- ADALINE (Adaptive Linear Neuron)
- Perceptron models (ADA_iris)
- Convolutional Neural Networks (CNN1, ConvNet)
- Recurrent Neural Networks (RNN4Births)
- Long Short-Term Memory networks (LSTM1, LSTM2)
- Autoencoders (AE_denoise)
- Hopfield Networks (HNN1, HNN2)

---

## 🎯 Purpose

This project is part of my machine learning learning path and helps me build intuition about:
